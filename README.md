# Tiny-NPU  

## 项目简介  
这是一个自研的轻量级AI芯片项目，重点围绕NPU的架构探索与验证展开。作为芯片验证工程师，通过该项目深入理解NPU核心业务逻辑，并构建可验证的硬件模型。  

## 规划功能  
- 基础张量运算  
- 验证测试平台  
- 简单RTL实现
