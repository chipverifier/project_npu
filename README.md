# project_npu

这是一个围绕神经处理单元（NPU）展开的开发项目，主要探索AI辅助与手动设计两种路径在NPU实现中的应用与对比。

## 项目背景
NPU作为AI处理特化的处理器，在边缘计算、实时图像处理等场景中展现出高效能优势。本项目旨在通过不同开发方式，实践NPU相关的架构设计、算法优化等核心能力。

## 版本说明
- **AI生成版本**：借助AI工具快速生成NPU相关代码框架、基础算法实现或文档初稿，聚焦开发效率与思路拓展。
- **手动操作版本**：通过人工编写、调试与优化，深入实现NPU的核心模块，侧重对技术细节的把控与底层原理的理解。

## 后续规划
项目将逐步完善各版本的功能实现，包括但不限于矩阵运算加速、神经网络推理支持等，并补充性能测试与对比分析。
